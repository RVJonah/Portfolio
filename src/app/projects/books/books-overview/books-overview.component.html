Project Overview

Books is a book review website which uses the goodreads API. It offers the ability for a user to search for a book using the ISBN, book title or author's name. The information is then returned combined with information from the goodreads API for average ratings and the number of ratings.

Books also offers it's own API service which will return the details of the book request using the api/<ISBN> route.

The site is currently being hosted on heroku and the full code can be found on my github.
