Project Aims

This project was the capstone of my experience with the Harvard CS50 computer science certificate.

The idea behind the project was to try to develop a web based application to help train Veterinarians and Veterinary Nurses in the various calculations that are required to be carried out by a qualified professional

I decided the project should have 3 goals:

  1. The function to teach a student the calculations necessary
  2. The option to answer practice questions without fear of failure
  3. The option to test and record a students own results
